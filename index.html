Bootstrap Template and Demo for a Project's Documentation

Taking modern features of Bootstrap and HTML5, this template aims to give a responsive web design to documentation sites, allowing them to be readable on any device, and not only a home computer.

But additionally as it is meant to be generic and reusable, it allows giving a homogeneous look to the documentation pages of various projects. It doesn't matter if they use Maven Site, Sphinx, or any other tool to build the doc site, just by adapting this template they will all show to be a documentation site.

Usage

Just grab the template from GitHub and adapt the html files with your preferred templating engine.

All the static files are stored in the _static folder, and should not need any modification.

Features

  • HTML5 semantic markup.
  • Using Bootstrap for a responsive design, which adapts to the device being used.
  • Icons created with Fonts Awesome.
  • Simple look which invites to explore the different options.
