Bootstrap Documentation for a Project

This template aims to give documentation sites a modern and responsive look, which will serve also to homogenize their looks.

That way it doesn't matter if the site has been created with Maven Site, Sphinx or any other tool, they all will have a same similar feeling, showing they are meant to be related.

Usage

Just grab the template from GitHub and adapt the html files with your preferred templating engine.

All the static files are stored in the _static folder, and should not need any modification.
